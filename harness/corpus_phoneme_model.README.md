# corpus_phoneme_model — corpus-derived phoneme cluster model

Generated by `harness/corpus_phoneme_model.py` (mg-ddee, harness v7). k-means over per-sign distributional fingerprints (left/right neighbors, position-in-inscription thirds, inscription genre). No substrate-pool prior.

## Build hyperparameters

- **bigram_smoothing**: `add-1 Laplace`
- **k**: `8`
- **kmeans_max_iters**: `100`
- **kmeans_seed**: `42`
- **model_version**: `v0`
- **n_records_used**: `428`
- **n_signs**: `136`
